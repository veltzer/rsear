# Getting Started

## Running

After installation, simply run:

```bash
rsear
```

A window opens displaying a piano keyboard and grand staff. The application automatically plays a C major scale followed by a C major chord, highlighting each note visually as it sounds.

## What You Will See

- **Grand staff** at the top — notes appear on the treble and bass clefs as they are played
- **Piano keyboard** below — keys light up in real time
- **Note names** displayed as text (e.g., "C4", "E4", "G4")

## Playback Sequence

1. **C major scale** — C3, D3, E3, F3, G3, A3, B3, C4 (each note held for 400ms)
2. **Brief pause** (200ms)
3. **C major chord** — C4, E4, G4 played simultaneously for 2 seconds
